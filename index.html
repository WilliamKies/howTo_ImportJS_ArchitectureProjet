Import & Export JS

Qu'est-ce que l'import en JavaScript ?

L'import en JavaScript est une fonctionnalité introduite dans ECMAScript 6 (également appelé ES2015) qui permet d'importer des fonctionnalités, des classes, des variables ou des modules entier d'un fichier JavaScript vers un autre. Cette fonctionnalité offre un moyen standardisé et efficace d'organiser, de réutiliser et de maintenir du code deans des projets JavaScript.

L'importation de modules JavaScript est particulièrement utile dans les grandes applications où le code doit être organisé en modules réutilisables et indépendants les uns des autres. Cette approche favorise la modularité, la maintenabilité et la réutilisabilité du code, ce qui est essentiel pour la gestion de projets complexes.

Quelques Points clés

  1. Organisation du code : L'importation de modules permet d'organiser le code en modules indépendants, ce qui facilite la gestion et la compréhension du code, en particulier dans les grands projets.
  2. réutilisabilité : Les modules importés peuvent être réutilisés dans différentes parties de l'application, ce qui permet d'éviter la duplication du code et de favoriser une approche DRY (Don't Repeat Yourself).
  3. Dépendances explicites : L'utilisation de l'importation de modules rend les dépendances entre les différents fichiers JavaScript explicites, ce qui facilite la compréhension des relations entre les différentes parties de l'application.
  4. Encapsulation : Les modules permettent d'encapsuler le code et de limiter la portée des variables, ce qui aide à éviter les collisions de noms et à assurer la sécurité du code.
  5. Interopérabilité : L'importation de modules est prise en charge par de nombreux outils et bibliothèques JavaScript modernes, ce qui facilite l'intégration avec d'autres technologies et frameworks.

Qu'est-ce que l'export en JavaScript ?

L'exportation de modules JavaScript est une fonctionnalité introduite dans ECMAScript 6 (également appelé ES2015) qui permet d'exporter des fonctionnalités, des classes, des variables ou des modules entiers depuis un fichier JavaScript vers un autre. Cette fonctionnalité est complémentaire à l'importation de modules et permet de rendre des éléments disponibles pour être importés dans d'autres fichiers JavaScript.

Quelques Points clés

  1. Exportation nommée : Vous pouvez exporter des fonctionnalités, des classes, des variables ou des modules entiers en utilisant l'instruction `export`. Vous pouvez exporter des éléments individuels en les spécifiant après l'instruction `export`, ou exporter un module entier en utilisant `export default`.
  2. Exportation par défaut : L'exportation par défaut est utilisée pour exporter un seul élément en tant qu'élément principal du module. Vous pouvez avoir au plus un export par défaut par module. Lors de l'importation d'un module avec exportation par défaut, vous pouvez lui donner le nom que vous voulez lors de l'importation.
  3. Exportation nommée multiple : Vous pouvez exporter plusieurs éléments nommés dans un même fichier JavaScript en les séparant par des virgules après l'instruction `export`.
  4. Réexportation : Vous pouvez réexporter des éléments d'autres modules en utilisant l'instruction `export` dans un fichier de réexportation. Cela permet de simplifier l'importation de plusieurs éléments depuis plusieurs fichiers en les regroupant dans un seul fichier de réexportation.

Démonstration de l'import / Export en JavaScript

Voici le code JS

script.js

import { addition, pi } from "./math.js";

function calcul() {
const nombre1 = parseFloat(document.getElementById('nombre1').value); const nombre2 = parseFloat(document.getElementById('nombre2').value); const resultat = addition(nombre1, nombre2); afficherDonnees(resultat); }

function afficherDonnees(resultat) {
const affichageDonnees = document.getElementById('affichageDonnees'); affichageDonnees.innerHTML = '';
const resultatElement = document.createElement('p'); resultatElement.innerHTML = `Le résultat est : ${resultat}
PI vaut : ${pi}`;
affichageDonnees.appendChild(resultatElement);
}

document.getElementById('calcul').addEventListener('click', function (event) {
event.preventDefault();
calcul();
});
math.js

export function addition(a, b) {
return(a + b);
}

export const pi = 3.14159;
